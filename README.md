# LUNAR EffecT

리뷰를 입력하면 랜덤으로 포인트를 증정하는 게임 이벤트 사이트

<br/>

# 개발기간

24.12.23 ~ 25.01.02

<br/>

# 소개

모바일 게임사의 이벤트 페이지를 컨셉으로하여, 리뷰 작성 시 포인트를 지급하는 시스템을 구현한 프로젝트입니다.

<br/>
